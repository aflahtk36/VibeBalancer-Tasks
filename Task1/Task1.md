## Branches

Branches are different versions of our project where we can work on changes without affecting the main code. Whenever we are working on a project we create a seperate branch, it is like a seperate line of development in your project . It allows us to work on new features without affecting the main code.

If we are working on a project, we create new branches for each new feature to be added.  
it is highly helpfull when we are doing opensource, multiple person can contribute to it  by creating new branches on their own, then they can merge this branch to the main branch , by creating a pull request the author of the main brach will rewiew his code and suggest any changes to be made then it is merged with the original project or main branch.
